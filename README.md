- Hero với avatar tròn
- Music player có playlist
- Gallery 6 ảnh theo thứ tự:

- About card bo tròn
- Hiệu ứng hoa anh đào, tim nhỏ, đốm sáng
- Lightbox toàn màn hình
- Responsive cho iPhone / Android
